json파일이 두개인지 비교

1개일떄
json파일 이용해서 일주일간 시청한 영상들의 총 시청시간, 날짜별 시청 시간(날짜과 시청시간 함께 나타내기), 평균 시청영상 갯수, 
날짜별 가장 많이 영상을 시청한 시간대를 분류하고 분류한 자료들을 이용해서 중독도 점수 매기기

점수 매기는 기준
일주일 총 시청시간 : 1점 <= 4시간 < 2점 <= 8시간 < 3점
평균 시청영상 갯수 : 1점 <= 50개 < 2점 <= 200개 < 3점
날짜별 가장 많이 영상을 시청한 시간대 : 0시 ~ 8시 사이면 1점 추가
매긴 중독도 점수랑 날짜별 시청시간을 앱인벤터로 보내기

앱인벤터에서 매긴 중독도 점수를 이용해서 점수를 정상 <= 4 < 주의 <=6 < 심각으로 나누기
날짜별 시청시간에 있는 자료를 이용해 그래프 만들기
중독도에 따라 해결방법 알려주기

2개일때
중독도가 나아졌는지 확인하는 기능
일주일 이상 전에 중독도와 해결방법을 확인했던 json파일과 오늘 다시 받은 json파일을 이용해 지난번에 비해 날짜별 시청시간이 얼마나 줄었는지,
늘었는지 그래프로 비교하고 중독가 지난번에 비해 어떻게 바뀌었는지 알려주고 해결방법 다시 제시
서버에서 비교할 json 파일들의 최근 시청 날짜를 비교해 최근 시청날짜가 일주일 이상 차이가 나지 않을 경우 파일을 계산하지 않고 앱인벤터에서
파일의 날짜 간격이 일주일 이상이 되어야 한다는 화면 띄우도록 메시지 보내기
